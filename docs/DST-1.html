이 노트북이 담고 있는 내용¶

  • klue 내 DST(Dialogue State Tracking) task를 이해하기 위함

  • DST task에 사용되는 데이터셋은 wos dataset이며, klue에서 한글버전으로 만들어짐

  • dst task를 구성하기 위해 어떻게 input data를 구성하는지 살펴봄

  • klue-base line 기준
